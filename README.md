# Udhar Tracker

A simple and useful web app for kirana stores and small shop owners in India to manage customer credit digitally — no more paper diary!

## Features

- Add customers with name, phone number, and amount
- Record new udhar entries and partial payments
- Dashboard showing total customers and total outstanding amount
- One-tap WhatsApp reminder sent directly to the customer
- All data saved locally in the browser — no login needed
- Works on mobile and desktop

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
- localStorage for data storage
- Deployed on Netlify

## Live Demo

[[Click here to view the app](https://udhar-tracker.netlify.app/)

## How to Run Locally

1. Clone this repository
2. Open `index.html` in your browser or use Live Server in VS Code

## Use Case

Built for small business owners in Maharashtra and across India who track customer credit manually in a paper diary. This app digitizes that process with zero technical knowledge required from the shop owner.
